# Long-Horizon Tasks
Long-horizon tasks are interactions that take place over extended conversations (typically 40–50 turns) where the agent must demonstrate deep historical recall and maintain temporal context throughout. The defining characteristic is that the agent needs to correctly retrieve and apply details mentioned much earlier in the conversation, without being explicitly reminded. For example, the agent should be able to identify a person, topic, or constraint introduced hours (or many turns) ago when prompted with something like "Now, send that file to the person we discussed this morning." These tasks can occur in any Universe (Stride, Meriton, Long Horizon, etc.) and are specifically designed to stress-test the model's ability to track context across a prolonged dialogue.
